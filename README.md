# kadath-rpg
KADATH v.0.2 Embárcate en un RPG retro inspirado en el universo de H.P. Lovecraft. Explora ciudades oníricas, gestiona tu inventario y combate criaturas de pesadilla mientras proteges tu cordura. Con un motor artístico estilo ASCII y sistema de guardado, ¿lograrás alcanzar la Ciudad Desconocida o sucumbirás al abismo? © Molvic Studio 2024.
